# Data Cleaning and Preprocessing - Task 1

## Objective
Clean and prepare a raw dataset by removing missing values, duplicates, and formatting inconsistencies.

## Tools Used
- Microsoft Excel

## Steps Performed
- Removed missing values using Excel filters.
- Removed duplicate rows using **Remove Duplicates**.
- Standardized text values (e.g., Gender → Male/Female).
- Cleaned column names (removed symbols, made consistent).
- Ensured correct data types: numbers, text, and categories.
- Added a new column **AGE GROUP** for better analysis.

## Results
- Final dataset contains **200 rows and 6 columns**.
- No missing or duplicate values.
- All columns are properly formatted and consistent.
